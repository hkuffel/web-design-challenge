Welcome


This website contains several visualizations that were created by analyzing weather data in a random sample of 500 cities. The data — which you can peruse by clicking "Data" above — comes originally via the OpenWeatherMap API. To generate my dataset, I randomly generated 1,500 latitude-longitude pairs, then used the CitiPy package to find the nearest city to each pair, if one exists. For each city, I then used the API to determine that city's temperature, wind speed, humidity and cloudiness percentage.


To view each visualization individually, you can choose one from the "Plots" dropdown up above. Alternatively, you can see all four at once by clicking "Comparisons."
